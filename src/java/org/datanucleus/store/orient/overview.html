DataNucleus "Orient" provides persistence of Java objects to Orient datastores. It builds on top of the DataNucleus "core" jar supporting specific features of Orient datastores. It provides querying capabilities using JDOQL, Orient native, or SQL. The code can be utilised with either JDO or JPA persistence APIs.

Support for this persistence layer is provided by way of a Free Forum (no guarantee of availability) or via Commercial Support from DataNucleus. Please refer to the Orient Website for details of Orient datastores.
